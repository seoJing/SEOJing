Repository는 DB에 접근하는 인터페이스입니다. Spring Data JPA를 쓰는 프로젝트에서는 보통 이렇게 생겼습니다. 처음 보면 구현 코드가 없어서 이상할 수 있습니다. 그런데 Spring Data JPA는 메서드 이름을 보고 쿼리를 만들어 주는 기능이 있습니다. 이 이름은 대략 이렇게 읽을 수 있습니다. 물론 모든 쿼리를 메서드 이름만으로 해결하지는 않습니다. 복잡한 경우에는 @Query를 쓰거나 QueryDSL 같은 도구를 쓰기도 합니다. 하지만 초반에는 Repository = DB 접근 통로라고 이해하면 충분합니다.
