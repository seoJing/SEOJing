<Subtitle level={2}>1. 배열 고차함수 — map, filter, reduce</Subtitle>

<Paragraph>
  고차함수(Higher-Order Function)는{" "}
  <strong>함수를 인자로 받거나 반환하는 함수</strong>입니다. 배열 메서드 중 가장
  중요한 세 가지를 살펴봅니다.
</Paragraph>

<Subtitle level={3}>map — 변환</Subtitle>

```js
const numbers = [1, 2, 3, 4, 5];

// 각 요소를 2배로
const doubled = numbers.map((n) => n * 2);
// [2, 4, 6, 8, 10]

// React에서 리스트 렌더링
const items = ["사과", "배", "포도"];
const listItems = items.map((item, index) => <li key={index}>{item}</li>);
```

<Subtitle level={3}>filter — 필터링</Subtitle>

```js
const products = [
  { name: "사과", price: 1000 },
  { name: "한우", price: 50000 },
  { name: "배", price: 2000 },
];

// 1만원 이하 상품만
const affordable = products.filter((p) => p.price <= 10000);
// [{ name: "사과", ... }, { name: "배", ... }]
```

<Subtitle level={3}>reduce — 누적</Subtitle>

```js
const cart = [
  { name: "사과", price: 1000, qty: 2 },
  { name: "배", price: 2000, qty: 1 },
];

// 총 금액 계산
const total = cart.reduce((acc, item) => acc + item.price * item.qty, 0);
// 1000*2 + 2000*1 = 4000

// reduce로 객체 만들기
const byName = cart.reduce((acc, item) => {
  acc[item.name] = item.price;
  return acc;
}, {});
// { 사과: 1000, 배: 2000 }
```

<Subtitle level={3}>체이닝 — 조합해서 사용</Subtitle>

```js
const scores = [85, 42, 93, 67, 55, 78, 91];

// 70점 이상인 점수들의 평균
const avg = scores
  .filter((s) => s >= 70) // [85, 93, 67, 78, 91]
  .reduce((sum, s, _, arr) => sum + s / arr.length, 0);
// 82.8
```

<Subtitle level={3}>그 외 자주 쓰는 배열 메서드</Subtitle>

```js
const arr = [1, 2, 3, 4, 5];

arr.find((n) => n > 3); // 4 — 첫 번째 일치 요소
arr.findIndex((n) => n > 3); // 3 — 첫 번째 일치 인덱스
arr.some((n) => n > 4); // true — 하나라도 조건 충족
arr.every((n) => n > 0); // true — 모두 조건 충족
arr.includes(3); // true — 포함 여부
arr.flat(); // 중첩 배열 1단계 펼치기
arr.flatMap((n) => [n, n * 2]); // map + flat 1단계
```

<Subtitle level={2}>2. 구조 분해 할당 — Destructuring</Subtitle>

<Paragraph>
  배열이나 객체에서 원하는 값을 변수로 한 번에 꺼내는 문법입니다. React에서
  props, useState, API 응답 처리 등 어디서나 사용합니다.
</Paragraph>

<Subtitle level={3}>배열 구조 분해</Subtitle>

```js
const [a, b, c] = [1, 2, 3];
// a=1, b=2, c=3

// 일부만 추출
const [first, , third] = [10, 20, 30];
// first=10, third=30

// 기본값
const [x = 0, y = 0] = [5];
// x=5, y=0

// React useState
const [count, setCount] = useState(0);
```

<Subtitle level={3}>객체 구조 분해</Subtitle>

```js
const user = { name: "철수", age: 25, role: "admin" };

const { name, age } = user;
// name="철수", age=25

// 이름 변경
const { name: userName } = user;
// userName="철수"

// 기본값
const { role = "user" } = {};
// role="user"

// 중첩 구조 분해
const {
  address: { city },
} = { address: { city: "서울" } };
// city="서울"

// React props
function Button({ label, onClick, disabled = false }) {
  return (
    <button onClick={onClick} disabled={disabled}>
      {label}
    </button>
  );
}
```

<Subtitle level={2}>3. 스프레드 연산자와 Rest 파라미터</Subtitle>

<Subtitle level={3}>스프레드 — 펼치기</Subtitle>

```js
// 배열 복사 & 합치기
const arr1 = [1, 2, 3];
const arr2 = [4, 5, 6];
const merged = [...arr1, ...arr2]; // [1, 2, 3, 4, 5, 6]

// 객체 복사 & 합치기 (React 상태 업데이트)
const state = { name: "철수", age: 25 };
const updated = { ...state, age: 26 };
// { name: "철수", age: 26 } — age만 변경된 새 객체

// 함수 인자
Math.max(...[1, 3, 2]); // 3
```

<Subtitle level={3}>Rest 파라미터 — 나머지 모으기</Subtitle>

```js
// 함수의 나머지 인자 모으기
function sum(first, ...rest) {
  return rest.reduce((acc, n) => acc + n, first);
}
sum(1, 2, 3, 4); // 10

// 구조 분해에서 나머지 모으기
const { name, ...others } = { name: "철수", age: 25, role: "admin" };
// name="철수", others={ age: 25, role: "admin" }

const [head, ...tail] = [1, 2, 3, 4, 5];
// head=1, tail=[2, 3, 4, 5]
```

<Subtitle level={2}>4. 옵셔널 체이닝(?.)과 null 병합 연산자(??)</Subtitle>

```js
const user = null;

// 기존 방식 — 장황함
const city = user && user.address && user.address.city;

// 옵셔널 체이닝 — 간결함
const city2 = user?.address?.city; // undefined (에러 없음)

// 메서드 호출에도 사용 가능
user?.greet?.();
```

```js
// null 병합 연산자 (??) — null/undefined일 때만 기본값
const name = user?.name ?? "익명";

// || 와의 차이
const count = 0 || "기본값"; // "기본값" — 0은 falsy
const count2 = 0 ?? "기본값"; // 0 — null/undefined만 처리
```

<Subtitle level={2}>5. ES6 모듈(Module)</Subtitle>

<Paragraph>
  모듈은 파일 단위로 코드를 분리하고, 필요한 것만 `import/export`하는
  시스템입니다. React 프로젝트는 전부 모듈 기반으로 작성됩니다.
</Paragraph>

```js
// math.js — named export
export const add = (a, b) => a + b;
export const subtract = (a, b) => a - b;

// Button.jsx — default export
export default function Button({ label }) {
  return <button>{label}</button>;
}
```

```js
// 다른 파일에서 import
import { add, subtract } from "./math.js";
import Button from "./Button.jsx"; // default는 중괄호 없이

// 이름 변경
import { add as sum } from "./math.js";

// 전부 가져오기
import * as math from "./math.js";
math.add(1, 2); // 3
```
